# 🏙️ 全国旅游攻略 (all-lv.com)

一站式中国旅游城市攻略网站 — 精选 **23 个省份 · 28 个城市**，景点/美食/攻略/行程一站搞定。

## 📋 项目概览

| 项目 | 说明 |
|------|------|
| **城市数量** | 28 个城市（覆盖全国主要旅游目的地及港澳） |
| **页面总数** | 约 250+ HTML 页面 |
| **技术栈** | 纯静态 HTML + CSS + JavaScript（无框架依赖） |
| **部署** | Vercel 静态站点 |
| **PWA** | 支持离线访问 |

## 🗂️ 项目结构

```
/
├── index.html              # 首页（省级分类 + 城市卡片）
├── style.css               # 全局样式设计系统
├── 404.html                # 404 页面
├── sitemap.xml             # 网站地图
├── robots.txt              # 爬虫规则
├── manifest.json           # PWA 清单
├── service-worker.js       # PWA 离线缓存
├── vercel.json             # Vercel 部署配置
├── city/{city}/            # 28 个城市页面
│   ├── index.html          # 城市首页
│   ├── attractions.html    # 景点推荐
│   ├── food.html           # 美食指南
│   ├── guide.html          # 实用攻略
│   ├── itinerary.html      # 行程规划
│   ├── blog.html           # 博客列表
│   └── blog/{slug}.html    # 博客详情页
├── data/cities.json        # 城市结构化数据
├── js/                     # JS 功能组件
├── css/                    # 组件样式
├── assets/                 # 图标资源
└── scripts/                # 生成脚本
```

## ✨ 功能特性

- ✅ **城市选择器** — 按省份分组，一键切换 28 个城市
- ✅ **全站搜索** — Ctrl+K 搜索景点/美食/攻略
- ✅ **天气预报** — 基于 Open-Meteo API 实时天气
- ✅ **行程规划** — 每个城市多日行程推荐
- ✅ **社交分享** — 微信/微博/复制链接
- ✅ **收藏评价** — localStorage 本地持久化
- ✅ **PWA 离线支持** — Service Worker 缓存
- ✅ **SEO 优化** — sitemap/OG/JSON-LD
- ✅ **响应式设计** — 全设备适配
- ✅ **无障碍访问** — 键盘导航 + ARIA
